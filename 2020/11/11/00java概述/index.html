java概述

1、java编译解析

源代码.java——字节码.class———机器码0101010

​ (通过编译器) （通过jvm虚拟机解析）

常见java程序：Application(应用程序)，Servlet(服务器端小程序)

2、java特性

面向对象、跨平台、健壮性

  • 跨平台性：在不同的操作系统只要安装一个java虚拟机，

​ 即面向UNIX的jvm、面向Windows的jvm、、、、

  • 健壮性：java是一种强类型语言，在编译和运行时进行大量的类型检查

​ java有自动垃圾回收功能GC，防止内存分配的错误

​ java有异常处理机制exception

  • JAVA SE:标准版 standard edition
  • JAVE EE:企业版 Enterprise Edition
  • JAVA ME:用于移动设备和嵌入式设备

3、JDK开发环境

（java开发工具）java的核心类库（JAVA API）、JRE 运行环境（jvm虚拟机+一些核心类库API）

4、dos命令环境

cd..返回上一级目录，dir当前目录，del删除文件

F:改变路径，cd/返回根目录，cd进入目录

md创建目录 ，rd删除目录 ，

ipconfig ip地址 ，systeminfo查看系统信息，

java -version java版本号

5、配置环境变量

  • 在path中编辑，%JAVA_HOME%\bin

  • 添加 变量名 JAVA_HOME,

    ​ 变量值：C:\Program Files (x86)\Java\jdk1.8.0_45

6、DOS命令

  • 通过javac hello.java命令调用javac.exe程序，生成字节码文件.class

  • 通过java hello命令，调用java.exe程序，使JVM解析字节码文件

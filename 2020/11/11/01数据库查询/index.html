数据库查询

发表于 2020-11-11 更新于 2020-12-17 分类于 MySql 阅读次数：

一、数据库DB

数据库：存储和管理数据的仓库，按照有关规则组织起来，可以被各种用户或应用共享的数据集合

数据库管理系统DBMS：用户通过数据库管理系统DBMS访问数据库中的数据

​ 特点：持久化存储数据，本质上是一个文件系统，关系型数据库通过的SQL语句进行管理DBMS

关系型数据库：MySQL

非关系型数据库:

二、MySQL

1、结构

一个数据库服务器包含多个库，一个数据库包含多个表，一个表包含多条记录，一条记录包含多个字段

2、SQL语句

（Structure Query Language）结构化查询语言 操作所有关系型数据

（1）分类

​ DDL：数据定义语言 create,drop,alter

​ DML（manipulation）：数据操作语言 insert delete update

​ DQL: 数据查询语言 select where

​ DCL: 数据控制语言 定义数据库的访问权限和安全级别以及创建用户 grant revoke

（2）数据类型

​ int double(4,1) time时间类型 date ( 年月日) datetime（年月日时分秒）

​ timestamp:时间戳类型 yyyy-MM-dd HH:mm:ss 记录某个业务日期

​ varchar:可变长度字符串 name varchar(20) :姓名最大20个字符

​ blob( binary large object) 大二进制

3、操作数据库

（1）创建

​ create database if not exists 数据库名 character set gbk; 不存在时创建数据库并指定字符集

（2）查询

show databases;

查询数据库的创建信息 show create database db1;

当前正在操作的数据库 select database();

使用命令切换当前正在使用的数据库 use 数据库名；

（3）修改删除

​ alter database 数据库名 character set GBK；

​ drop database if exists 数据库名；

4、操作表

（1）DDL

​ a、创建：create table 表名（

​ 列名 1 数据类型1，

​ 列名2 数据类型2 , //最后一列不加逗号

​ score DOUBLE(4,1),

​ sname VARCHAR(50)

​ ）；

​ 创建一个相似的表：create table 表名2 like 表名；

​ b、查看 : 查看表结构 desc 表名

​ 查看当前数据库的所有表 show tables;

​ c、修改： 修改表名 alter table 表名 rename to 新表名

​ alter table 表名 add remark varchar(30);

​ alter table 表名 change remark intro varchar(20);

​ alter table 表名 modify intro varchar(130);

​ d、删除

​ 删除一列：alter table student4 drop intro

​ 删除表: drop table if exists student4;

（2）DML

a、insert

​ 1、列名和值一致

​ 2、除了数字类型，其他类型需要使用引号引起来

​ 3、默认值null

（1）所有字段都赋值

​ INSERT INTO student3(id,sname,age,score,birthday,mark) VALUES(1001,’张三’,20,90.5,’2000-11-10’,”英语”)

（2）部分字段赋值

​ INSERT INTO student3(id,sname,age) VALUES (1000,’李四’,12);

（3）省略字段名

​ INSERT INTO student3 VALUES(1001,’张三’,NULL,90.5,’2000-11-10’,”英语”)

（4）一次添加多条记录

INSERT INTO student3 **VALUES(**1001,’张三’,20,90.5,’2000-11-10’,”英语”),(1001,’张三’,20,90.5,’2000-11-10’,”英语”)

b、delete

​ delete from xx where score>=90;

​ 不加条件，则删除表中所有记录

​ truncate table student; 一次删除 delete from xx; 一次记录删除一次

c、update

​ UPDATE student3 SET score=90,age=22,birthday=’1998-05-12’ WHERE id=1002;

（3）DQL

select from where group by having order by limit

select：查询（列）

不会对数据库中的数据进行修改，从数据库发送结果给客户端，是虚拟的表

查询表格字段 SELECT *****FROM stu;

查询指定列 SELECT sname,age FROM stu;

消除重复 select distinct job from emp;

计算列 SELECT ename AS 姓名,sal AS 薪资,comm AS 奖金,comm+sal AS 月收入 FROM emp;

IFNULL（comm,0）：将null视为0 SELECT comm,IFNULL(comm,0) FROM emp;

列的别名 SELECT ename AS 姓名,sal AS 薪资,comm AS 奖金,IFNULL(comm,0)+sal AS 月收入 FROM emp;

行列同时筛选 SELECT id,studentname,age FROM student WHERE age>=20;

条件查询where （行）

SELECT FROM student WHERE age>=20;
SELECT FROM student WHERE age<>**20; – 不等于20
SELECT *FROM student WHERE age!=20;

not

SELECT *FROM student WHERE NOT age=20;

and
SELECT *FROM student WHERE age>20 AND age<30;
SELECT *FROM student WHERE age>=20 && age<30;

between and

SELECT *FROM student WHERE age BETWEEN 20 AND 30;

or

SELECT *FROM student WHERE age=20 OR age=18 OR age =45;

in

SELECT *FROM student WHERE age IN(20,18,45);

is

SELECT *FROM student WHERE english IS NULL;

SELECT *FROM student WHERE english IS not NULL;

（4）通配符

下划线 _：代表一个字符

百发号**%**:代表至少一个字符

查询第二个字为德的同学 SELECT *FROM student WHERE studentname LIKE ‘_德%’;

（5）聚合函数

选择非空的列

count(列名)、max 、min 、sum、avg

查询员工人数 count(*)

SELECT COUNT(empno) FROM emp;

SELECT COUNT(*) FROM emp;

查询工资总额 SELECT SUM(sal) FROM emp;

平均工资 SELECT AVG(sal) FROM emp;

最大工资 SELECT MAX(sal) FROM emp;

总月收入

SELECT SUM(sal+IFNULL(comm,0)) FROM emp;

SELECT SUM(sal)+SUM(comm)FROM emp;

（6）分组查询

group by之后查询的字段，分组字段、聚合函数

a、单组

查询男女各自的人数 SELECT gender AS 性别,COUNT(*) AS 人数 FROM student GROUP BY gender;

按照性别分组，分别查询男、女同学的数学平均分，人数，要求:分数低于70，不参与分组
SELECT gender,AVG(math),COUNT(*) FROM student WHERE math>=70 GROUP BY gender;

b、having和where

a、where 在分组前进行限定，having在分组后进行限定

b、where 后不可以跟聚合函数，having可以进行聚合函数的判断

按照性别分组，分别查询男、女同学的数学平均分，人数，要求:分数低于70，不参与分组
SELECT gender,AVG(math),COUNT(*) FROM student WHERE math>=70 GROUP BY gender;

查询各个部门的人数，只显示人数大于8个的部门
SELECT deptno,COUNT(*) FROM emp GROUP BY deptno HAVING COUNT（ * ）>8;

查询每个部门的部门编号以及每个部门工资大于1500的人数
SELECT deptno,COUNT(*) FROM emp WHERE sal>1500 GROUP BY deptno;

c、多组分组

查询某个专业的某门课程的总分,平均

SELECT speciality,coursename,SUM(score),AVG(score) FROM scores,course,studentinfo WHERE
scores.courseno=course.courseno AND scores.studentno=studentinfo.studentno
GROUP BY speciality,coursename;

（7）排序

a、单组

order by 排序字段 排序方式

descend降序 ascend升序

按年龄升序排序,默认升序 SELECT FROM student ORDER BY age ASC;
按年龄降序排序 SELECT *FROM student *ORDER BY** age DESC;

b、多组排序

*年龄降序，相同年龄按数学成绩降序
SELECT *FROM student *ORDER BY age DESC,math DESC;

（8）分页

limit

SELECT *FROM emp LIMIT 0,3; – 第一页 (pageno-1)*pagesize
SELECT *FROM emp LIMIT 3,3; – 第二页
SELECT *FROM emp LIMIT 6,3;
SELECT *FROM emp LIMIT 9,3;
SELECT *FROM emp LIMIT 12,3;（8）顺序

select from where group by having order by limit

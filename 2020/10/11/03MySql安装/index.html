Linux安装MySQL

1、下载MySql安装包

https://dev.mysql.com/downloads/mysql/5.7.html#downloads

2、安装

2.1 卸载原有mysql

[root@localhost ~]# rpm -qa|grep -i mysql
perl-DBD-MySQL-4.023-5.el7.x86_64
[root@localhost ~]# rpm -e perl-DBD-MySQL-4.023-5.e17.x86_64

2.2 上传MySql安装包

在远程端：alt+p

put E:/test/MySQL-5.6.22-1.el6.i686.rpm-bundle.tar(安装包的绝对路径)

2.3 解压MySql的安装包

创建存放目录：mkdir MySql

解压：tar -xvf MySQL-5.6.22-1.el6.i686.rpm-bundle.tar -C /root/mysql

2.4 安装MySql依赖

安装：yum -y install libaio.so.1 libgcc_s.so.1 libstdc++.so.6 libncurses.so.5 –setopt=protected_multilib=false

更新：yum update libstdc++-4.4.7-4.el6.x86_64

2.5 安装服务端、客户端

服务端：rpm -ivh MySQL-client-5.6.22-1.el6.i686.rpm

客户端：rpm -ivh MySQL-server-5.6.22-1.el6.i686.rpm

问题

无法启动客户端server：

  1. wget命令：yum -y install wget

  2. 创建存放安装文件：mkdir mysql-server , cd mysql-server

  3. 下载mysql的repo包：wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm

  4. 安装 ：rpm -ivh mysql-community-release-el7-5.noarch.rpm

    ​ yum install mysql-server

3、启动MySql服务

3.1 启动

启动数据库：systemctl start mysqld.service

登录：mysql -u root -p 密码：默认为空

3.2 修改权限

连接数据库：修改权限，刷新

grant all privileges on . to ‘root’ @’%’ identified by ‘password’;

flush privileges;

3.3 存在连接问题解决

无法连接问题解决

  • 关闭Linux防火墙：systemctl stop firewald.service
